# family

## Definition

The "family" is the set of all `chirag127/oriz*` repositories: 11
public sites, an open-ended count of Chrome/Firefox/Edge extensions,
6 shared npm packages, and one Hono Worker API — all governed by the
shared `AGENTS.md` rules.

## Expanded

The 11 current sites are `oriz-home`, `oriz-blog`, `oriz-books`,
`oriz-book-lore`, `oriz-cards`, `oriz-finance`, `oriz-journal`,
`oriz-image-tools`, `oriz-pdf-tools`, `oriz-urls-to-md` (planned to
fold into `oriz-dev-tools`), and `oriz-me`. The 6 packages live under
`@chirag127/*`: `oriz-kit`, `firebase-init`, `auth-ui`, `contact-form`,
`sidebar`, `oriz-family`. The single API (`apps/api/`) is a Hono
Worker deployed at `api.oriz.in`.

Every member of the family inherits the four non-negotiables and the
five hard rules from `AGENTS.md`.
