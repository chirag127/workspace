# oriz

## Definition

`oriz` is simultaneously the brand, the master GitHub repo name
(`chirag127/oriz`), and the apex domain (`oriz.in`) under which every
site in the family lives as a subdomain.

## Expanded

The brand has three faces. As a **GitHub identity**, `chirag127/oriz`
is the umbrella repo that holds every site, extension, and package as
a submodule. As a **domain**, `oriz.in` is the apex; every site is a
subdomain (`blog.oriz.in`, `me.oriz.in`, `auth.oriz.in`, …). As a
**word**, it prefixes every repo name (`oriz-blog`, `oriz-me`,
`oriz-kit`) so the family is greppable across GitHub.

The four non-negotiables of the brand: impress recruiters, serve as
Chirag's lifelong archive, cost zero to host forever, and never expose
a credit-card billing surface.
