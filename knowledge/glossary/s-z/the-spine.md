# the spine

## Definition

The Spine is `oriz-blog`'s typographic series indicator — a row of
filled and hollow circles connected by horizontal rules
(e.g. `●─●─○─○`) that shows where a post sits in a multi-part series.

## Expanded

Filled circles (`●`) are posts already published; hollow circles
(`○`) are forthcoming. The horizontal rules connecting them are the
"spine". It renders inline, in the post header, and in the series
index page.

The Spine is `oriz-blog`'s most distinctive visual element and is
explicitly typographic — no SVG, no images, just unicode and CSS —
so it survives every render context including the GitHub Pages
survival fallback and RSS readers.
