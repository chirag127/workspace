# Microsoft Clarity

## Role

Records anonymized user sessions and renders heatmaps so we can see
how people actually use each site.

## Free tier

- Unlimited sessions
- Unlimited heatmaps
- Unlimited projects
- No retention cap mentioned at our scale

## Card / subscription required?

**NO.** Sign-in with Microsoft / Google / Facebook account; no
payment method requested.

## Alternatives

- LogRocket (1K sessions free)
- OpenReplay self-hosted

## Swap cost

Low — replace `<script>` tag.

## Why this is our pick

Truly unlimited, run by Microsoft so it's not going away, and the
heatmaps are good.
