# Speed stack — three orthogonal compression layers

## Why

Latency in agentic coding has three sources:
1. **Input** — tokens sent to the model (files, chat history, tool output)
2. **Output** — tokens generated by the model
3. **Workflow** — interruptions (rate limits, errors)

Each layer has a different compressor. Stacking them is compounding, not redundant.

## The stack (free, no fork edits)

| Layer | Tool | Compresses | Install scope | Conflicts with fork? |
|---|---|---|---|---|
| Input — chat/files | Headroom | File reads, long chat history | Global proxy at `:8787` | No (proxy layer) |
| Input — shell | RTK (Rust Token Killer) | `git diff`, `npm install`, `ls -R` output | Global `~/.claude/` | No (tool wrapper) |
| Output | Ponytail | Code generation (lazy-senior-dev rule ladder) | Claude Code plugin | No (system-prompt skill) |

## What we explicitly do NOT install

- **9Router / OmniRoute** — these REPLACE the proxy at `:8787`. We already
  run Headroom→hai→Bedrock there (SAP-mandated). Switching breaks that.
- **Caveman** — no installable skill exists in 2026-06. The terse-output
  effect is achievable via Ponytail's system prompt directly.
- **NeuralMind** — vaporware coming-soon landing page as of 2026-06.

## Install scope

All three tools live entirely OUTSIDE any fork:
- Headroom: Docker container at `:8787` (proxy)
- RTK: Rust binary in `~/.cargo/bin/` + config in `~/.claude/`
- Ponytail: Claude Code plugin in `~/.claude/plugins/`

**Zero merge-conflict surface with forks.** Per `fork-minimize-conflict-surface`,
this is what we want — the speed stack is agent infrastructure, not fork code.

## Install command (auto)

```cmd
C:\D\oriz\scripts\install-speed-stack.cmd
```

Idempotent. Re-runs converge. Per `automate-never-runbook` — never a manual list.

## Anti-patterns

- ❌ Install Caveman by copy-pasting a system prompt into every fork's CLAUDE.md
- ❌ Replace the Hr→hai chain with 9Router without re-verifying SAP routing
- ❌ Install NeuralMind / paid SaaS memory layers (we have OKF knowledge tree)
- ❌ Run RTK + 9Router simultaneously (both intercept tool output)
